# 2.4  競合上のポジショニングを行う

## 概要と理由
ユーザーがエージェントに「{あなた} 対 {競合}」や「{競合} の代替」を尋ねると、エージェントは見つけられる限りのページを返します。あなたが比較を書いていなければ、上位の結果はあなたの競合のブログか、アフィリエイト収益のために最適化された第三者のレビューサイト - 正確さのためではなく - になります。比較のサーフェスを自ら所有することが、エージェントにあなたの差別化についての引用可能なファーストパーティ版を持たせる方法です。同じ論理が **価格** にも当てはまります。「{あなた} はいくらか?」と尋ねられたエージェントには、ファーストパーティで機械可読な答えが必要であり、さもなければ古びた第三者の推測を引用します - そして価格は、買い手が成約前に尋ねる最も意図の強い事柄の1つです。

## スコアリング
- **工数 2/5** - ほとんどが執筆作業です。主要な競合ごとに1ページ、加えて代替をまとめた集約ページ。
- **インパクト 4/5** - 「対」クエリは、B2B における意図の強いエージェントトラフィックの非常に大きな割合を占めます。
- **視覚的変化: 大** - 新規の公開マーケティングページが増えます。`/compare/{competitor}`、`/alternatives`、`/pricing`。

## 手順
1. **上位3〜5の名指しの競合について `/compare/{competitor}` ページを公開する。** 各ページに: 1段落の正直な要約、機能比較表、価格モデルの比較、1〜2の顧客成果指標（「{競合} から切り替えた後、顧客は6か月で、買い物客に表示される在庫切れエラーが31%減少したと報告している」- あるいは、あなたの買い手が実際に気にするコンバージョン、フルフィルメント、リテンションの数値が何であれそれ）。表は `Schema.org/Table` で、ページは `Article` の JSON-LD でマークアップします。
2. **単一の `/alternatives` 集約ページを公開する。** 「{あなたの製品} の代替」- 各競合を簡潔にカバーし、それぞれがどんなときにより適しているか（そう、あなたが最適ではない場合も含めて）を示し、競合ごとのページへリンクします。エージェントは知的誠実さを引用で報います。
3. **エージェントが読める `/pricing` ページを公開する。** *（手順3〜4は、ビジネスモデルに公開された価格表がある場合 - デジタル商材、サブスクリプション、SaaS 隣接のコマース - に当てはまります。多くのトランザクション型小売業者は、プランの階層ではなく製品ページで SKU ごとに価格を付けます。それがあなたなら、価格はすでに `Product` / `Offer` の JSON-LD（[2.1](./m2-1-json-ld.md)）に存在しているので、手順5に飛ばして構いません。）* すべてのプラン階層、その価格、課金単位、含まれるもの - 画像や JS でレンダリングされるウィジェットではなく、実在する HTML テキストと `<table>` として。各階層を `schema.org/Offer` の JSON-LD（`price`、`priceCurrency`、`name`）でマークアップし、[2.1](./m2-1-json-ld.md) の `Product` / `SoftwareApplication` スキーマの下にネストします。価格が本当に従量制なら、計算式と実例を示してください - 「曖昧、お問い合わせください」は、価格がないのと同じに読まれます。
4. **機械可読な `/pricing.md` を追加する。** 料金ページのプレーンな Markdown ミラー - 階層ごとに、価格、単位、上限の1セクション - を `text/markdown` として提供します。これは、エージェントがコストの質問に1往復で答えるために取得するファイルであり、`llms.txt`（[2.2](./m2-2-llms-txt-content.md)）の `## 料金` セクションと対になります。
5. **トーンは事実に基づき、勝ち誇らない。** 「{競合} はイベント単位の価格設定で、当社は {あなたの単位} に紐づく成果単位の価格設定です」は、「{競合} の価格設定は分かりにくく高い」に勝ります。前者は引用可能で、後者はマーケティングノイズとしてふるい落とされます。
6. **出典を引用する。** すべての競合に関する主張は、競合自身のドキュメント、料金ページ、または日付入りの公開声明にリンクすべきです。出典のない主張は、取得時に格下げされます。
